# Controls
## Movement
* **Left Arrow**: move left
* **Right Arrow**: move right
* **Z**: jump
## Attack
* **X**: basic attack
* **C**: ultimate attack
* **Up Arrow + X/C**: attack upward
* **Down Arrow + X/C**: lower attack
